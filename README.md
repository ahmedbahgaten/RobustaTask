# Introduction 
KMS is a Cross-Plateform Mobile app that is written in Xamarin using Xamarin.Forms 

# Minimum iOS OS Version 
We support iOS 13+ 

# Minimum Android OS Version
We support Android 5.0 (API level 21) +

# Technology
Xamarin.Forms 5.0.0.2083
